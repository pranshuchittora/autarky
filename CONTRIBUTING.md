### Up & Running

- #### Fork

- #### Clone

```
git clone https://github.com/{YOUR_USERNAME}/autarky.git
```

- #### Install deps _(Prefer `yarn`)_

```bash
# npm
npm i

# yarn
yarn
```

- #### Start dev mode

```bash
# npm
npm run watch

# yarn
yarn run watch
```

- #### Running the code

```bash
# Development build
node /dist/index.js

# Production Build
node /build/index.js
```

- #### Testing
  We are using `jest` as a testing frammework

```bash
# npm
npm run test

# yarn
yarn run test
```
